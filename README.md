# Privacy Policy

## Quran Mushaf (مصحف القرآن الكريم) — developed by SoftIslam

**Quran Mushaf**, published by **SoftIslam**, does not share any user identifiable information with third party entities. Quran Mushaf does use Google’s Fabric service for anonymously tracking analytics and crashes, but all of this data is not tied back to a certain person.
